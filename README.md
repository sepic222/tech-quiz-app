# TechUp! ⚡

**Level up your tech & business vocabulary — one quiz at a time.**

TechUp! is a lightweight, gamified flashcard quiz app for anyone working in or around the tech industry. No account needed, no install — just open and play.

→ **[Play it live](https://sepic222.github.io/tech-quiz-app/)**

---

## What's inside

**7 topic categories**

| Category | What you'll learn |
|---|---|
| Tech Vocabulary | APIs, CI/CD, Docker, SSH, and the rest |
| AI & LLMs | RAG, tokens, context windows, AI agents |
| SaaS Metrics | ARR, NRR, CAC, PLG, and what they actually mean |
| Sales Methods | BANT, discovery calls, champions, deal stages |
| Product Management | Sprints, roadmaps, KPIs, MVP, scope creep |
| Biz Dev & GTM | Pipeline, ICP, ACV, QBR, SLA |
| Terminal Basics | cd, ls, git clone, grep, and friends |

**3 quiz modes**

- **Multiple Choice** — pick the right term from 4 options
- **Flashcards** — flip the card, self-rate your confidence
- **Type It** — type the answer from memory (hardest)

**Progression system** — earn XP, build daily streaks, and master terms by getting them right 3 times.

---

## Running it locally

No build step, no dependencies. Just clone and open.

```bash
git clone https://github.com/sepic222/tech-quiz-app.git
cd tech-quiz-app
open index.html   # or double-click the file
```
